# **App Name**: LexConnect Admin Panel

## Core Features:

- User Registration and Profiling: Allows registration of users (opportunities) with profile data storage (name, contact info, documents). Integrates with existing ERP as CRM opportunity.
- Volunteer Registration: Enables registration of volunteers and stores their information.
- Branch/Authorized Point Management: Manages registration and information for branches/authorized points.
- Case Management: Facilitates the creation, tracking, and management of cases (Amparo Constitucional, etc.) with categorization (contenciosa/no contenciosa).
- Internal Notification System: Provides an internal notification system for updates on opportunities and cases.
- Opportunity Lifecycle Tracking: Tracks the lifecycle of opportunities from registration to case conversion.
- Profile Update History: Stores historical profile data for auditing and consistency in case records, maintaining non-destructive updates.

## Style Guidelines:

- Primary color: Deep blue (#103365), instills trust and authority.
- Background color: Very light blue (#ddd9d9), provides a calm and professional backdrop.
- Accent color: Purple (#cc0000), draws attention to important actions and notifications.
- Font pairing: 'Space Grotesk' for headlines and 'Inter' for body text. 'Space Grotesk' is a sans-serif and 'Inter' is a sans-serif.
- Use professional, clear icons from a consistent set (e.g., Material Design Icons) to represent entities, actions and statuses.
- Implement a responsive grid layout with clear sections for navigation, data display, and action controls.
- Subtle transition animations for page loads, form submissions, and data updates, indicating system activity.
